******** Présentation

Cette application a pour but de faciliter la gestion des comptes clients de votre banque.

Ainsi, aux fonctionnalités déjà présentes dans la version 1 s'ajoutent ici la possibilité de se connecter à une base de données (MySQL) afin d'y stocker, modifier ou supprimer des datas, mais aussi une toute nouvelle interface web grâce à laquelle il sera plus aisé pour vos conseillers d'effectuer toutes ces tâches.

ProxiBanqueSI, une application proche de ses utilisateurs pour une banque plus proche de ses clients.



******** Contenu du livrable

Vous trouverez avec cette présentation :

- un diagramme de classes présentant la structure principale de l'application
- un diagramme de cas d'utilisation rappelant les fonctionnalités offertes aux utilisateurs
- un diagramme de déploiement expliquant les installationss matérielles que recquiert l'utilisation de l'application
- une documentation présentant les classes et méthodes utilisées
- la base de données proxibanque.sql dans laquelle sont contenues les données clients
- l'archive ProxiBanqueSI.war
